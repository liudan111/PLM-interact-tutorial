Welcome to PLM-interact's documentation!
===================================
**PLM-interact** is a Python library for PPI prediction and protein mutation analysis.

We trained models from two PPI benchmarking PPI datasets, and human PPIs create by us which is 

pulls data from the `Open Food Facts database <https://world.openfoodfacts.org/>`_


Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
